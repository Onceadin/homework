Отчет Лаб 1 Сокол Андрей
1. Задание:
Программа, которая считывает с потока ввода символ разделителя, потом принимает строку, которую разбивает на подстроки используя символ разделителя и выводит результат на стандартный поток вывода в обратном порядке.
2. Программа была реализованна в редакторе NotePad++
3. String sep = args0; - считываю первый аргумент.
4. String[] text = args1.split(sep); - разбиваю строку на подстроки, используя символ разделителя.
5. Для вывода в обратном порядке используется простой цикл for.
6. Код программы был скомпилированн с помощью команды:
javac -d bin src/default/MainClass.java
7. Для запуска программы использувалась команда:
java -classpath ./bin MainClass arg0 arg1
